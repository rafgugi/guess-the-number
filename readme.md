## Guess the number

### How do I get set up?

- Ensure you have [Node.js](https://nodejs.org/en/), [npm](https://www.npmjs.com/), [bower](https://bower.io/), and [gulp](http://gulpjs.com/)
- Run `npm install`
- Run `bower install`
- Run `npm start`
- Open `index.html` in browser

What? Just play the game.
